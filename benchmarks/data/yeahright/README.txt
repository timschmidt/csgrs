Yeah Right
==========
contact: Keenan Crane (kcrane@uiuc.edu)

Files
--------------------------------------------
This archive contains the "Yeah Right," model including:

1. the original Catmull-Clark control mesh
2. quad and triangle tessellations of the surface at various resolutions

All meshes are stored as Wavefront OBJ, including the Catmull-Clark control mesh.
Note that for the control mesh your implementation must support control
polygons with an arbitrary number of vertices (some commercial packages support
only triangles and quads).  The surface is orientable, has genus 131, and has a
manifold embedding.
